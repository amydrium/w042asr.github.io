Greatest Success

When I planned my Genius Hour web page, I had not anticipated what I would need to learn for it to function. The page looks simple now, though I went through many stages of trying to figure out more complex styling.I will talk about the various styling methods I learned throughout this experience. On my website, I had learned the necessary code for a complex image embed. Not to mention, I first created my complex navigation bar on my Genius Hour page.

When I was nearing the end of research period of my Genius Hour project, I found that the site I used to track my progress had an option to create an embedded Instagram page. I thought this would work well enough, but it did not work on our schoolâ€™s restricted Wi-Fi network. To remedy this, I wanted to make the exact same thing, but in a way that worked without Iframes. I wanted to make an exact copy of the embedded webpage I used previously. To do this, I had to learn, on my own time, to add an image overlay on hover, to add size-responsive equidistance (so the space between sections was equal at all times, regardless of screen size.), and most difficult of all, to scale with the rest of my webpage. I have a similar image showcase area on my about page, which had the same problem; when the website scaled up or down, the formatting would break. I have now fixed the issue, though I had not yet learned this when I created my Genius Hour page. Though this knowledge was not actually applied to my Genius Hour page, I still learned from it. If not for learning about responsive image formatting, I never would have been able to fix my about me page.
